# tiny-sql-js

A tiny in-memory database written in Node.js

## Requirements

- Node >= 18.x
- NPM / Yarn

## Installation

First, install the dependencies with

```sh
yarn
```

Then, build it running

```sh
yarn build
```

Start the app with

```sh
yarn start
```
